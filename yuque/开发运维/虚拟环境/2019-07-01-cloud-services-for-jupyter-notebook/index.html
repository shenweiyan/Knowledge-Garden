    зјәе°‘зҡ„еҠҹиғҪпјҡ CoCalcВ зӣ®еүҚдёҚж”ҜжҢҒдәӨдә’ејҸе°ҸйғЁд»¶гҖӮ

    • CoCalc еҢ…еҗ«дёҖдёӘз§°дёәж—¶й—ҙж—…иЎҢзҡ„ејәеӨ§зүҲжң¬жҺ§еҲ¶еҠҹиғҪпјҢеҸҜд»ҘзІҫзЎ®и®°еҪ•з¬”и®°жң¬дёӯзҡ„жүҖжңүжӣҙж”№пјҢе№¶е…Ғи®ёжӮЁдҪҝз”Ёзӣҙи§Ӯзҡ„ж»‘еқ—жҺ§д»¶жөҸи§Ҳиҝҷдәӣжӣҙж”№гҖӮ
    • CoCalc жҜҸйҡ”еҮ еҲҶй’ҹе°ұдјҡдҝқеӯҳжүҖжңүйЎ№зӣ®ж–Үд»¶зҡ„еӨҮд»ҪпјҢиҝҷж„Ҹе‘ізқҖжӮЁеҸҜд»Ҙж №жҚ®йңҖиҰҒжҒўеӨҚж—§зүҲжң¬зҡ„ж–Үд»¶гҖӮ
    • CoCalc еҢ…жӢ¬ж•ҷеӯҰзҡ„дёҖдәӣе…¶д»–еҠҹиғҪпјҢдҫӢеҰӮеҲҶеҸ‘е’ҢиҜ„еҲҶдҪңдёҡзҡ„иғҪеҠӣпјҢд»ҘеҸҠеңЁеӯҰз”ҹе·ҘдҪңе’ҢдёҺеӯҰз”ҹеҲҶдәӨжөҒдә«д»»еҠЎж—¶и§ӮеҜҹеӯҰз”ҹзҡ„иғҪеҠӣгҖӮ

    ж•°жҚ®йӣҶжҳ“з”ЁжҖ§пјҡ жӮЁеҸҜд»Ҙд»Һжң¬ең°и®Ўз®—жңәе°Ҷж•°жҚ®йӣҶдёҠдј еҲ°йЎ№зӣ®дёӯпјҢе№¶дё”йЎ№зӣ®дёӯзҡ„д»»дҪ•з¬”и®°жң¬йғҪеҸҜд»Ҙи®ҝй—®иҜҘж•°жҚ®йӣҶгҖӮ

    **зҪ‘з»ңи®ҝй—®пјҡ**дёҚпјҢдҪҝз”Ёе…Қиҙ№и®ЎеҲ’ж—¶ж— жі•дҪҝз”ЁжӯӨеҠҹиғҪгҖӮ

    **дҝқиҜҒе·ҘдҪңз§ҒеҜҶжҖ§зҡ„иғҪеҠӣпјҡ**е…Ғи®ёгҖӮ

    е…¬ејҖеҲҶдә«зҡ„иғҪеҠӣпјҡ е…Ғи®ёгҖӮеҰӮжһңжӮЁйҖүжӢ©е…¬ејҖжӮЁзҡ„з¬”и®°жң¬е№¶еҲҶдә«й“ҫжҺҘпјҢеҲҷд»»дҪ•дәәйғҪеҸҜд»ҘеңЁдёҚеҲӣе»ә CoCalc еёҗжҲ·зҡ„жғ…еҶөдёӢи®ҝй—®иҜҘз¬”и®°жң¬пјҢе№¶дё”жӢҘжңү CoCalc еёҗжҲ·зҡ„д»»дҪ•дәәйғҪеҸҜд»Ҙе°Ҷе…¶еӨҚеҲ¶еҲ°иҮӘе·ұзҡ„еёҗжҲ·гҖӮ

    еҚҸдҪңиғҪеҠӣпјҡ е…Ғи®ёгҖӮжӮЁеҸҜд»Ҙе°Ҷз¬”и®°жң¬дҝқеҜҶпјҢдҪҶеҸҜд»ҘйӮҖиҜ·зү№е®ҡдәәе‘ҳиҝӣиЎҢзј–иҫ‘гҖӮжӮЁе’ҢжӮЁзҡ„еҚҸдҪңиҖ…еҸҜд»ҘеҗҢж—¶зј–иҫ‘з¬”и®°жң¬пјҢе№¶е®һж—¶жҹҘзңӢеҪјжӯӨзҡ„жӣҙж”№пјҲе’Ңжёёж ҮпјүпјҢд»ҘеҸҠеңЁз¬”и®°жң¬ж—Ғиҫ№зҡ„зӘ—еҸЈдёӯиҒҠеӨ©пјҲдҪҝз”Ёж–Үжң¬жҲ–и§Ҷйў‘пјүгҖӮжүҖжңүи®Ўз®—зҡ„зҠ¶жҖҒе’Ңз»“жһңд№ҹжҳҜеҗҢжӯҘзҡ„пјҢиҝҷж„Ҹе‘ізқҖжүҖж¶үеҸҠзҡ„жҜҸдёӘдәәйғҪе°Ҷд»ҘзӣёеҗҢзҡ„ж–№ејҸдҪ“йӘҢз¬”и®°жң¬гҖӮ

    е…Қиҙ№и®ЎеҲ’зҡ„жҖ§иғҪиЎЁзҺ°пјҡ жӮЁе°ҶеҸҜд»ҘдҪҝз”Ёе…·жңү 1 GB е…ұдә« RAM е’Ң 3 GB зЈҒзӣҳз©әй—ҙпјҲжҜҸдёӘйЎ№зӣ®пјүзҡ„ 1 ж ёе…ұдә« CPUгҖӮдјҡиҜқе°ҶеңЁ 30 еҲҶй’ҹдёҚжҙ»еҠЁеҗҺе…ій—ӯпјҢдҪҶе®ғд»¬еҸҜд»ҘиҝҗиЎҢй•ҝиҫҫ 24 е°Ҹж—¶гҖӮ

    жҖ§иғҪеҚҮзә§пјҡ ж”ҜжҢҒгҖӮжӮЁеҸҜд»Ҙдёә CoCalc ж”Ҝд»ҳи®ўйҳ…иҙ№з”ЁпјҢиө·д»·дёәжҜҸжңҲ 14 зҫҺе…ғгҖӮжҲ–иҖ…пјҢжӮЁеҸҜд»ҘеңЁиҮӘе·ұзҡ„и®Ўз®—жңәдёҠе®үиЈ… В CoCalc Docker й•ңеғҸпјҢиҝҷж ·жӮЁе°ұеҸҜд»Ҙе…Қиҙ№иҝҗиЎҢеӨҡдёӘз§Ғжңүз”ЁжҲ·зҡ„ CoCalc жңҚеҠЎеҷЁгҖӮ

    ж–ҮжЎЈе’ҢжҠҖжңҜж”ҜжҢҒпјҡ CoCalc жӢҘжңүеӨ§йҮҸж–ҮжЎЈгҖӮз”ЁжҲ·еҸҜйҖҡиҝҮз”өеӯҗйӮ®д»¶е’ҢиҒ”зі»еҲ—иЎЁиҺ·еҫ—ж”ҜжҢҒпјҢе№¶еңЁ В GitHubВ  дёҠи·ҹиёӘдә§е“Ғй—®йўҳгҖӮ

    з»“и®әпјҡ дҪҝз”Ё CoCalc зҡ„жңҖжңүеҗёеј•еҠӣжҳҜе®ғзҡ„е®һж—¶еҚҸдҪңе’Ң В “time travel”В  зүҲжң¬жҺ§еҲ¶еҠҹиғҪпјҢд»ҘеҸҠиҜҫзЁӢз®ЎзҗҶеҠҹиғҪпјҲеҰӮжһңжӮЁжҳҜж•ҷз»ғпјүгҖӮ иҷҪз„¶з•ҢйқўжңүзӮ№ж··д№ұпјҢдҪҶзҺ°жңүзҡ„ Jupyter з”ЁжҲ·еҸҜд»ҘзӣёеҜ№иҪ»жқҫең°иҝҮжёЎеҲ° CoCalcгҖӮдҪҶжҳҜпјҢе…Қиҙ№и®ЎеҲ’зЎ®е®һеӯҳеңЁдёҖдәӣйҮҚиҰҒзҡ„йҷҗеҲ¶пјҲж— жі•е®үиЈ…е…¶д»–иҪҜд»¶еҢ…жҲ–и®ҝй—® InternetпјүпјҢеҸҰеӨ–пјҢе…Қиҙ№и®ЎеҲ’зҡ„жҖ§иғҪд№ҹдёҚй«ҳгҖӮ

    6. Datalore¶

    DataloreВ  жҳҜз”ұ JetBrainsпјҢдёҖдёӘеҲ¶дҪң PyCharmпјҲдёҖз§ҚжөҒиЎҢзҡ„ Python IDEпјүзҡ„е…¬еҸёеҲӣе»әзҡ„гҖӮе®ғзҡ„е…Ҙй—Ёе°ұеғҸеҲӣе»әеёҗжҲ·жҲ–дҪҝз”Ё Google жҲ– JetBrains еёҗжҲ·зҷ»еҪ•дёҖж ·з®ҖеҚ•гҖӮжӮЁеҸҜд»ҘеҲӣе»әж–°зҡ„ Datalore “е·ҘдҪңз°ҝ(workbook)” жҲ–дёҠдј зҺ°жңүзҡ„ Jupyter NotebookгҖӮ Datalore е·ҘдҪңз°ҝд»Ҙдё“жңүж јејҸеӯҳеӮЁпјҢдҪҶе®ғж”ҜжҢҒеҜје…Ҙе’ҢеҜјеҮәж ҮеҮҶ В .ipynbВ  ж–Үд»¶ж јејҸгҖӮ

    **ж”ҜжҢҒзҡ„иҜӯиЁҖпјҡ**еҸӘж”ҜжҢҒ Python 3гҖӮ

    е®үиЈ…иҪҜд»¶еҢ…пјҡ DataloreВ  йў„е®үиЈ…дәҶж•°зҷҫдёӘиҪҜд»¶еҢ…пјҢжӮЁеҸҜд»ҘдҪҝз”Ё В pipВ  жҲ– В condaВ  е®үиЈ…е…¶д»–иҪҜд»¶еҢ…пјҢд№ҹеҸҜд»ҘйҖҡиҝҮжҢҮе®ҡиҪҜд»¶еҢ…зҡ„ GitHub еӯҳеӮЁеә“жқҘе®үиЈ…гҖӮ

    з•ҢйқўзӣёдјјеәҰпјҡ еҪ“дҪ жү“ејҖ Datalore ж—¶пјҢз•ҢйқўзЎ®е®һзұ»дјјдәҺ Jupyter з¬”и®°жң¬пјҢеӣ дёәжңүд»Јз Ғе’Ң Markdown еҚ•е…ғж јд»ҘеҸҠеңЁиҝҷдәӣеҚ•е…ғж јдёӢж–№зҡ„иҫ“еҮәгҖӮдҪҶжҳҜпјҢDatalore е’Ң Jupyter з•Ңйқўд№Ӣй—ҙеӯҳеңЁдёҖдәӣйҮҚиҰҒе·®ејӮпјҡ

    • еҚ•е…ғж јпјҲDatalore з§°д№Ӣдёә “blocks”пјүдёҚдјҡиў«зј–еҸ·пјҢеӣ дёәеҚ•е…ғж је·Із»Ҹиў«ејәеҲ¶жҖ§иҝӣиЎҢдәҶжҺ’еәҸгҖӮ жҚўеҸҘиҜқиҜҙпјҢжүҖжңүд»Јз ҒйғҪеҝ…йЎ»жҢүз…§жңҖз»ҲеёҢжңӣе®ғиҝҗиЎҢзҡ„йЎәеәҸзј–еҶҷгҖӮ
    • з¬”и®°жң¬пјҲDatalore з§°д№Ӣдёә “workbook”пјүеҸҜд»ҘжңүеӨҡдёӘе·ҘдҪңиЎЁпјҢзұ»дјјдәҺ Google иЎЁ ж јпјҢиҝҷжҳҜе°Ҷй•ҝе·ҘдҪңз°ҝеҲҶи§ЈдёәйҖ»иҫ‘йғЁеҲҶзҡ„дҫҝжҚ·ж–№ејҸгҖӮеҰӮжһңеңЁе·ҘдҪңз°ҝдёӯеҲӣе»әеӨҡдёӘе·ҘдҪңиЎЁпјҢеҲҷжүҖжңүе·ҘдҪңиЎЁе…ұдә«зӣёеҗҢзҡ„зҺҜеўғгҖӮеӣ дёәеҚ•е…ғж јйЎәеәҸеңЁ Datalore дёӯеҫҲйҮҚиҰҒпјҢжүҖд»Ҙз¬¬дәҢдёӘе·ҘдҪңиЎЁдёӯзҡ„еҚ•е…ғж јиў«и§ҶдёәеңЁз¬¬дёҖдёӘе·ҘдҪңиЎЁдёӯзҡ„еҚ•е…ғж јд№ӢеҗҺпјҢз¬¬дёүдёӘе·ҘдҪңиЎЁеңЁз¬¬дәҢдёӘе·ҘдҪңиЎЁд№ӢеҗҺпјҢдҫқжӯӨзұ»жҺЁгҖӮ
    • иҝҳжңүи®ёеӨҡе…¶д»–з•Ңйқўе·®ејӮпјҢиҝҷдәӣе·®ејӮеңЁ “add features” йғЁеҲҶдёӯиҝӣиЎҢдәҶи§ЈйҮҠгҖӮ

    й”®зӣҳеҝ«жҚ·й”®пјҡ й”®зӣҳеҝ«жҚ·й”®еҸҜз”ЁдәҺ Datalore дёӯзҡ„еӨ§еӨҡж•°ж“ҚдҪңпјҢдҪҶеҝ«жҚ·ж–№ејҸдёҺ Jupyter дҪҝз”Ёзҡ„еҝ«жҚ·ж–№ејҸжҲӘз„¶дёҚеҗҢгҖӮ

    зјәе°‘зҡ„еҠҹиғҪпјҡ

    • Datalore дёҚдҪҝз”Ё IPython еҶ…ж ёпјҢеӣ жӯӨ IPython йӯ”жңҜеҮҪж•°е’Ң shell е‘Ҫд»ӨдёҚеҸҜз”ЁгҖӮпјҲдҪҶжҳҜпјҢеҜ№ IPython еҶ…ж ёзҡ„еҸҜйҖүи®ҝй—®зҡ„еҠҹиғҪе·Із»ҸеңЁи®ЎеҲ’дёӯгҖӮпјү
    • з”ұдәҺ Datalore иҸңеҚ•ж ҸдҝқжҢҒйқһеёёз®ҖеҚ•е№¶дё”жІЎжңүе·Ҙе…·ж ҸпјҢеӣ жӯӨи®ёеӨҡж“ҚдҪңеҸӘиғҪдҪҝз”Ёй”®зӣҳеҝ«жҚ·й”®е®ҢжҲҗгҖӮ
    • жӮЁж— жі•е°Ҷе·ҘдҪңз°ҝдёӢиҪҪдёәе…¶д»–жңүз”Ёзҡ„ж јејҸпјҢдҫӢеҰӮ Python и„ҡжң¬пјҢHTML зҪ‘йЎөжҲ– Markdown ж–Үд»¶гҖӮ
    • Datalore дёҚж”ҜжҢҒе…¶ Markdown еҚ•е…ғж јдёӯжүҖжңүеёёз”Ёзҡ„ Markdown еҠҹиғҪгҖӮпјҲдҪҶжҳҜпјҢж”№иҝӣзҡ„ Markdown ж”ҜжҢҒеҠҹиғҪе·Із»ҸеңЁи®ЎеҲ’дёӯгҖӮпјү
    • Datalore дёҚж”ҜжҢҒдәӨдә’ејҸе°ҸйғЁд»¶гҖӮ
    • Datalore дёҚеҢ…жӢ¬еӨҡдё»жңәж”ҜжҢҒгҖӮ

    ж–°еўһзҡ„еҠҹиғҪпјҡ

    • еҚ•е…ғж јеңЁжӮЁзј–еҶҷж—¶иҮӘеҠЁиҝҗиЎҢпјҢDatalore з§°д№Ӣдёә “е®һж—¶и®Ўз®—”гҖӮиҝҷе®һйҷ…дёҠдҪҝжӮЁеңЁзј–еҶҷд»Јз Ғж—¶жӣҙе®№жҳ“и°ғиҜ•д»Јз ҒпјҢеӣ дёәжӮЁеҸҜд»Ҙз«ӢеҚіжҹҘзңӢд»Јз Ғзҡ„з»“жһңгҖӮпјҲеҸҜд»ҘзҰҒз”Ёе®һж—¶и®Ўз®—пјҢеңЁиҝҷз§Қжғ…еҶөдёӢпјҢжӮЁеҸҜд»ҘжүӢеҠЁи§ҰеҸ‘иҰҒиҝҗиЎҢзҡ„еҚ•е…ғж јгҖӮпјү
    • з”ұдәҺеҚ•е…ғж је°Ҷе§Ӣз»ҲжҢүе…¶жҺ’еҲ—йЎәеәҸиҝҗиЎҢпјҢеӣ жӯӨ Datalor e еҸҜд»Ҙи·ҹиёӘеҚ•е…ғж јдҫқиө–жҖ§гҖӮиҝҷж„Ҹе‘ізқҖеҪ“зј–иҫ‘з»ҷе®ҡеҚ•е…ғж јж—¶пјҢDatalore е°ҶзЎ®е®ҡе…¶дёӢйқўзҡ„е“ӘдәӣеҚ•е…ғеҸҜиғҪеҸ—еҲ°еҪұе“ҚпјҢе№¶е°Ҷз«ӢеҚійҮҚж–°иҝҗиЎҢиҝҷдәӣеҚ•е…ғж јпјҲеҒҮи®ҫеҗҜз”ЁдәҶе®һж—¶и®Ўз®—пјүгҖӮеҰӮжһңзј–иҫ‘еңЁд»ҺеұһеҚ•е…ғж јдёӯеҜјиҮҙй”ҷиҜҜпјҢеҲҷдјҡз«ӢеҚіж Үи®°иҝҷдәӣй”ҷиҜҜгҖӮ
    • Datalore е…Ғи®ёжӮЁжҢүйЎәеәҸжҳҫзӨәеҚ•е…ғж јиҫ“е…Ҙе’Ңиҫ“еҮәпјҲе°ұеғҸеңЁ Jupyter дёӯдёҖж ·пјүжҲ– “жӢҶеҲҶи§Ҷеӣҫ(split view)”пјҢеңЁиҝҷз§Қжғ…еҶөдёӢпјҢиҫ“е…Ҙе’Ңиҫ“еҮәдҪҚдәҺдёӨдёӘеҚ•зӢ¬зҡ„зӘ—ж јдёӯгҖӮдҪҝз”ЁйЎәеәҸи§Ҷеӣҫж—¶пјҢDatalore иҝҳеҸҜд»ҘиҪ»жқҫйҡҗи—ҸжүҖжңүиҫ“е…ҘжҲ–йҡҗи—ҸжүҖжңүиҫ“еҮәгҖӮ
    • еңЁд»Јз Ғе®ҢжҲҗдёӯпјҢDatalore еҢ…еҗ«жҜ” Jupyter жӣҙеӨҡзҡ„”жҷәиғҪ”гҖӮ
    • еңЁзј–еҶҷд»Јз Ғж—¶пјҢDatalore дјҡдёәжӮЁеҸҜиғҪжғіиҰҒйҮҮеҸ–зҡ„ж“ҚдҪңжҸҗдҫӣдёҠдёӢж–Үж„ҹзҹҘе»әи®®пјҲз§°дёә”ж„Ҹеӣҫ(intentions)”пјүгҖӮдҫӢеҰӮпјҢеңЁй”®е…Ҙ DataFrame зҡ„еҗҚз§°д№ӢеҗҺпјҢж„ҸеӣҫеҸҜиғҪеҢ…жӢ¬ “drop string columns”гҖҒ”histogram” е’Ң “train test split”гҖӮеҪ“жӮЁеҚ•еҮ»ж„Ҹеӣҫж—¶пјҢDatalore е®һйҷ…дёҠдјҡдёәжӮЁз”ҹжҲҗд»Јз ҒпјҢиҝҷеҸҜиғҪжҳҜеӯҰд№ жҹҗдәӣд»»еҠЎиғҢеҗҺзҡ„д»Јз Ғзҡ„жңүз”Ёж–№жі•гҖӮ

    **зҪ‘з»ңи®ҝй—®пјҡ**ж”ҜжҢҒгҖӮ

    жҖ§иғҪеҚҮзә§пјҡ жҡӮдёҚж”ҜжҢҒпјҢиҷҪз„¶еҫҲеҝ«е°ұдјҡжңүдёҖдёӘд»ҳиҙ№и®ЎеҲ’пјҢе®ғжҸҗдҫӣжӣҙеӨҡзҡ„зЈҒзӣҳз©әй—ҙе’ҢжӣҙејәеӨ§зҡ„ CPUпјҲжҲ– GPUпјүгҖӮ

    еҰӮдҪ•йҖүжӢ©еҗҲйҖӮзҡ„жңҚеҠЎ¶
